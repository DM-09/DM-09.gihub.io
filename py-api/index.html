소개

py-api는 API를 불러올 때 CORS에러가 나는 것을 방지해 주는 프록시 서비스 입니다.


사용

https://api-py.vercel.app/?r=[URL]
위 [URL] 부분에 API URL을 넣으시면 됩니다.

구버전 URL도 작동 합니다




테스트


주의 사항

  • Get Method만 지원합니다
  • 파이썬을 사용하여 시간이 조금 소요됩니다
  • 예러가 나면(잘못된 URL 등) 'Error'를 반환합니다
